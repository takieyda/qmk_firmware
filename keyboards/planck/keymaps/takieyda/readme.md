# Takieyda's Planck Layout

More to come.

## Description
A layout that takes the default Planck layout and adds a little bit from the Pok3r.

## Features
- Layers
  - Base
  - Lower
  - Raise
  - Nav: Hold Esc to access volume controls and VIM style arrow keys.
  - Adjust: Hold Lower and Raise
